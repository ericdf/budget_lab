# Berkeley Budget Lab

**https://ericdf.github.io/budget_lab/**

A prioritization and communication tool for Berkeley residents facing real budget decisions.

## What it does

Anchors users in the City Manager's actual budget scenarios and forces explicit prioritization:

- Select a scenario — **tax passes** (moderate reductions) or **tax fails** (deeper reductions)
- Assign each city department to **A** (protect), **B** (accept baseline cut), or **C** (absorb more)
- See immediately how protecting some services shifts the savings requirement onto others
- Draft and send a prioritization email to City Council via your own email client

## Core insight

Protecting some services requires other services to absorb deeper reductions. The tool makes that tradeoff visible and helps residents communicate it clearly.
